# Code Review Command

このコマンドは、変更されたコードに対して包括的なコードレビューを実行します。

## 実行内容

1. **Git変更の確認**
   - `git status`でステージングされた変更を確認
   - `git diff`で具体的な変更内容を取得

2. **レビュー観点**
   以下の観点からコードをレビューします：

   - **コード品質**: 可読性、保守性、一貫性
   - **ベストプラクティス**: Next.js/Reactのベストプラクティスへの準拠
   - **TypeScript**: 型安全性、型定義の適切性
   - **パフォーマンス**: 不要な再レンダリング、最適化の機会
   - **セキュリティ**: 潜在的なセキュリティ問題
   - **アクセシビリティ**: a11yのベストプラクティス

3. **フィードバック形式**
   - 問題点と改善提案を明確に記載
   - 重要度（Critical/High/Medium/Low）を付与
   - 具体的なコード例を提示

## 使用方法

```bash
/review
```

変更をコミットする前にこのコマンドを実行してください。
